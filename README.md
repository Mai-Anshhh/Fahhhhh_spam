# Fahhhhh_spam
An application that spams "fahhhhh" meme over and over.
It closes upon clicking 'x' button but respawns after 3 seconds again, you kill the task through task manager.
This is just meant to be a small prank and I made this only for learning purposes (no wrong intents)

A demo:

https://github.com/user-attachments/assets/49ff4261-2643-43c8-998c-d92e55826e44

# Use 
Just install the fahhhhh.exe from the repo, and run that.
To close it simply kill the process from your task manager.
